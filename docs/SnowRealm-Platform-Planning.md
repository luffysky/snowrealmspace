# SnowRealm Platform 整合規劃（給 Claude Code）

## 背景

目前 SnowRealm 不再只是單一產品，而是整合成一個 Platform。

產品： - AI Island - SnowRealm Space - Insight - YukiBoard - MD2Deck -
TamonRadar - 毛行天下

目標不是產品數量，而是產品彼此正相關、互補、互相增強。

## 核心理念

-   每個產品都能提升其他產品價值。
-   新產品加入要讓整個平台更強。
-   共用 Platform 能力，不重複造輪子。

任何新產品至少符合： 1. 使用既有 Platform 能力。 2.
回饋至少一個既有產品。

## Platform

SnowRealm = Platform 產品只是 Platform 上的應用。

共用： - SSO - Theme - Agent - AI - Memory - Notification - Search -
Analytics - Storage

## SnowRealm+

不要做各產品 Pro。 改成 SnowRealm+。

會員可使用整個平台能力。

## Economy

### AI Dot

AI 運算資源。 用途： - AI聊天 - AI生成 - AI分析 - Agent - 翻譯 - 摘要 -
寫程式

特性： - 每月重置 - 可加購 - 與 Z 幣分離

### Z 幣

平台貨幣。

用途： - 主題 - 貼圖 - Marketplace - 課程 - Agent 模板 - 數位商品

特性： - 可累積 - 任務取得 - 創作者販售獲得

## 新手流程

建立帳號： - 可登入 - 可瀏覽 - 僅送約 50 AI Dot 體驗

更多 AI Dot： - Email 驗證 - 手機驗證 - 新手教學 - 第一個 Agent - 第一個
AI 任務

## Trust Level

Level0：註冊 Level1：Email Level2：Email+手機 Level3：SnowRealm+

## 任務

AI Dot： - Email - 手機 - 教學 - Agent

Z幣： - 每日登入 - 完成課程 - 分享 - Marketplace - 活動

## AI Router

統一管理： - Provider - 模型 - Token - 成本 - AI Dot

依成本與難度自動切換模型。

## Storage

使用 Cloudflare R2。 Platform 控制 quota。 所有產品共用 Storage。

## 長期目標

-   一個帳號
-   一個會員
-   一套 AI Dot
-   一套 Z 幣
-   一套 Platform
-   全產品互補、互相增強

請以 Platform 優先，而不是單一產品優先，重新規劃整體架構。
